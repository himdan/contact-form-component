# Contact Form Application with Docker Compose

A full-stack contact form application with web component frontend, Node.js backend, and PostgreSQL database.

## PreRequistes
1. makefile
2. docker

## Quick Start

1. **Clone and navigate to project:**
   ```bash
   git clone <repository-url>
   cd contact-form-app
   cp .env.sample .env
   make build
   make up
