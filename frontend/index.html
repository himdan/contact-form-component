Contact Form Application

Modern web component with Docker Compose deployment

About This Application

This is a fully Dockerized contact form application featuring:

  • Web Component frontend
  • Node.js REST API backend
  • PostgreSQL database
  • Nginx web server
  • Automatic deployment with Docker Compose
  • Real-time form validation
  • Responsive design
Docker Web Components Node.js PostgreSQL Nginx Express

API Endpoints

POST /api/contacts - Submit contact form

GET /api/contacts - List all contacts

GET /api/health - Health check
